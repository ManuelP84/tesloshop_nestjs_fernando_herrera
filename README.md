# Teslo API

1. Clone the project

2. ```npm install``` 

3. Clone the file ```.env.template``` and rename it to ```.env```

4. Change the environment variables

5. Run the data base
```
docker-compose up -d
```

6. Run (on development):
```npm run start:dev```
