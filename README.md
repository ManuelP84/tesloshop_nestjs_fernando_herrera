# Teslo API

1. Clone the project

2. Run the command to instal the packages
```
npm install
``` 

3. Clone the file ___.env.template___ and rename it to ___.env___

4. Change the environment variables

5. Run the data base
```
docker-compose up -d
```

6. Run (on development):
```
npm run start:dev
```

7. Execute SEED

```
http://localhost:3000/api/seed
```
